Detailed Description

A class for the binned observable \(<A_{6c}>\) in \(M \to V l^+l^-\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the binned observable \(<A_{6c}>\) in \(M \to V l^+l^-\) in terms of the binned CP average helicity coefficients \(<\Delta_i>\), computed in the MVll class:

\[ <A_{6c}>= \frac {<\Delta_{6c}>}{<\Gamma'>} \,. \]
