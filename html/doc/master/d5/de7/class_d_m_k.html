Detailed Description

A class for \(\Delta M_{B_s}\), the difference in mass of between the heavy and light mass eigenstates of the neutral \(B_d\) system.

A class for \(\Delta M_K\), the difference in mass of between the heavy and light mass eigenstates of the neutral Kaon system.

This class is used to compute the theoretical value of \(\Delta M_{B_s}\). The short distance contributions to this parameter is calculable both in the SM and in NP models. In the SM the short distance contribution is dominated by the top loop in the box diagram.

This class is used to compute the theoretical value of \(\Delta M_K\). While the short distance contributions to this parameter is calculable both in the SM and in NP models, the long distance contribution can only be estimated and might contribute to as much as half the value of the parameter. This long distance contribution is put in as an input parameter in this code through SM.getDMK().
