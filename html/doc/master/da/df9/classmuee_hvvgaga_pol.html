Detailed Description

A class for computing the ratio \(\){e^+e^- H, H } \(\), excluding contributions from on-shell \(Z\to \nu\bar{\nu} \).

A class for computing the ratio \(\mu_{e^+e^- \to \nu\nu H, H \to \gamma\gamma}\) between the \( e^{+}e^{-}\to \nu\bar{\nu} H, H \to \gamma\gamma \) production cross-section in the current model and in the Standard Model.
