Detailed Description

An observable class for the left-right asymmetry in \(e^+e^-\to Z\to e^+ e^-\) at the \(Z\) pole.

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the left-right asymmetry for an \(e^+e^-\to Z\to e^+ e^-\) process at the \(Z\) pole. Lepton-flavour universality is not assumed.
