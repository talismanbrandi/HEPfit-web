Detailed Description

An observable class for the effective \(H \mu\mu\) coupling \(g_{H\mu\mu}^{Eff}\), defined from the square root of \(\Gamma_{H\mu\mu}/\Gamma_{H\mu\mu}^{SM}\).

This class is used to compute the effective \(H \mu\mu\) coupling \(g_{H\mu\mu}^{Eff}\), defined from the square root of \(\Gamma_{H\mu\mu}/\Gamma_{H\mu\mu}^{SM}\).
