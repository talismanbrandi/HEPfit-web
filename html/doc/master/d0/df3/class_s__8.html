Detailed Description

A class for the binned observable \(<S_8>\) in \(M \to V l^+l^-\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the binned observable \(<S_8>\) in \(M \to V l^+l^-\) in terms of the binned CP average helicity coefficients \(<\Sigma_i>\), computed in the MVll class:

\[ <S_8>= \frac {<\Sigma_{8}>}{<\Gamma'>} \,. \]
