Detailed Description

A class for the Bernoulli numbers.

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class handles the Bernoulli numbers, which are used to compute the trilogarithm function Polylogarithms::Li3() and the Clausen function of index three ClausenFunctions::Cl3. A list of the Bernoulli numbers can be found in [116].
