Detailed Description

A model class for new physics in the form of an electroweak chiral Lagrangian with a light Higgs-like scalar.

Author
HEPfit Collaboration
Copyright
GNU General Public License

This is a Model class describing new physics models based on the electroweak chiral Lagrangian with a light neutral scalar \(h\). The effective Lagrangian expansion is truncated at the level of two-derivatives [61],

\begin{eqnarray} {\cal L} &=& \frac {1}{2} (\partial_\mu h)^2 - V(h) + \frac{v^2}{4}\mathrm{Tr}(D_\mu \Sigma^\dagger D^\mu \Sigma) \left(1+2a\frac{h}{v}+b\frac{h^2}{v^2}+\cdots\right) \\ && - \left[ m_{u,i}\overline{Q}_{L,i}\Sigma \left(\begin{array}{c}u_{R,i}\\0\end{array}\right)\left(1+c_u \frac{h}{v}+\cdots\right) + m_{d,i}\overline{Q}_{L,i}\Sigma \left(\begin{array}{c}0\\d_{R,i}\end{array}\right)\left(1+c_d \frac{h}{v}+\cdots\right) + m_{e,i}\overline{L}_{L,i}\Sigma \left(\begin{array}{c}0\\e_{R,i}\end{array}\right)\left(1+c_e \frac{h}{v}+\cdots\right) + \mathrm{h.c.} \right], \end{eqnarray}

where \(V(h)\) is given by

\[ V(h) = \frac{m_h}{2}h^2 + \frac{d_3}{6}\left(\frac{3m_h^2}{v}\right)\,h^3 + \frac{d_4}{24}\left(\frac{3m_h^2}{v^2}\right)\,h^4 + \cdots\,. \]

The SM corresponds to the choice \(a=b=c_u=c_d=c_e=d_3=d_4=1\). The dominant deviations from the SM in the electroweak precision observables are induced by the non-standard \(HVV\) coupling \(a\neq 1\). This generates extra contributions to the oblique parameters [28] ,

\begin{eqnarray} S &=& \frac{1}{12\pi} (1 - a^2) \ln\bigg(\frac{\Lambda^2}{m_h^2}\bigg)\,,\\ T &=& - \frac{3}{16\pi c_W^2} (1 - a^2) \ln\bigg(\frac{\Lambda^2}{m_h^2}\bigg)\,, \end{eqnarray}

where \(\Lambda\) is the cutoff scale of the model.

Initialization

After creating an instance of the current class with the constructor NPHiggs(), it is required to call the initialization method InitializeModel(). In the Monte Carlo run, the constructor as well as the initialization method are called in InputParser::ReadParameters().

Model parameters

The model parameters of NPHiggs are summarized below:

Label LaTeX symbol Description
a \(a \) The \(hVV\) coupling \(a\).
b \(b \) The \(hhVV\) coupling \(b\).
c_u \(c_u \) The \(h\overline{u}u\) coupling \(c_u\).
c_d \(c_d \) The \(h\overline{d}d\) coupling \(c_d\).
c_e \(c_e \) The \(h\overline{e}e\) coupling \(c_e\).
d_3 \(d_3\) The \(hhh\) coupling \(d_3\).
d_4 \(d_4\) The \(hhhh\) coupling \(d_4\).
LambdaNP \(\Lambda\) The cutoff scale of the effective Lagrangian. When the value of this parameter is set to 0 in model initialization, the cutoff scale is taken to be \(\Lambda = 4\pi v/\sqrt{|1-a^2|}\), where the \(W_LW_L\) scattering becomes non-perturbative.

Model flags

There is no model flag in the current class.
