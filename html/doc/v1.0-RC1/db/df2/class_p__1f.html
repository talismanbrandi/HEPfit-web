Detailed Description

A class for the observable \(P_1\) in \(M \to V l^+l^-\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the observable \(P_1\) in \(M \to V l^+l^-\) in terms of the CP averaged helicity coefficients \(\Sigma_i\), computed in the MVll class:

\[ P_1=\frac {\Sigma_3}{2\Sigma_{2s}}\,. \]
