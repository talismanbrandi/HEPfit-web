1 HEPfit - a Fitting Tool for the Standard Model and Beyond
2 ===================================================================
3 
4 version v1.0-RC1
5 -----------
6 
7 HEPfit is a flexible tool which, given the Standard %Model or any extension,
8 allows to:
9 
10  - fit the model parameters to a given set of experimental observables
11  - obtain fit results for observables
12  - obtain predictions for observables
13 
14 The HEPfit library can be used:
15 
16  - to perform a Bayesian Markov Chain Monte Carlo analysis of the implemented model using BAT support
17  - to perform a Bayesian Markov Chain Monte Carlo analysis of user defined models and/or observables using BAT support
18  - to perform a user defined statistical analysis with the models and observables implemented in the library
19  - to perform a user defined statistical analysis with user defined models and/or observables
20 
21 The Markov Chain Monte Carlo is based on the [BAT (Bayesian Analysis Toolkit) library](https://www.mppmu.mpg.de/bat/).
