Model parameters

The model parameters of THDM are summarized below. The current implementation only allows for a \(Z_2\) symmetric Two-Higgs-Doublet model without CP violation in the Higgs potential. The scalar 125 GeV resonance is assumed to be the lighter CP-even Higgs state of the model.

Label LaTeX symbol Description
logtb \(\log_{10} \tan \beta\) The decadic logarithm of the tangent of the mixing angle \(\beta \equiv \arctan \frac{v_2}{v_1}\).
bma \(\beta - \alpha\) The difference of the mixing angles \(\beta\) and \(\alpha\).
mHh2 \(m_H^2\) The mass square of the heavier CP-even Higgs state.
mHh2mmA2 \(m_H^2-m_A^2\) The difference of the squared masses of the heavier CP-even Higgs and the CP-odd Higgs.
mHh2mmHp2 \(m_H^2-m_{H^+}^2\) The difference of the squared masses of the heavier CP-even Higgs and the charged Higgs.
m12_2 \(m_{12}^2\) The soft \(Z_2\) symmetry breaking parameter.
bsgamma_theoryerror \(\theta_{b\to s\gamma}^{\text{theo}}\) A nuisance parameter between -1 and +1 for the theoretical error in the determination of BR(B X_s ).

Model flags

Label Value Description
modelTypeflag typeI / typeII / typeX / typeY This flag determines the type of \(Z_2\) symmetry.
