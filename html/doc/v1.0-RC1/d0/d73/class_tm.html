Detailed Description

A class for the form factor \(<T_->\) in \(B \to K^*\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the averaged form factor \(<T_->\) in \(B \to K^*\) using LCSR results at low \(q^2\) and lattice results at high \(q^2\)
