Detailed Description

A class for the observable \(A_{FB}\) in \(M \to V l^+l^-\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the observable \(A_{FB}\) in \(M \to V l^+l^-\) in terms of the CP averaged helicity coefficients \(\Sigma_i\), computed in the MVll class:

\[ A_{FB}=- \frac {3\Sigma_{6s}}{4\Gamma'} \,. \]
