Detailed Description

An observable class for \(R_\ell^0=\Gamma(Z\to {\rm hadrons})/\Gamma(Z\to \ell^+ \ell^-)\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the ratio of the \(Z\)-boson hadronic width to the \(Z\to \ell^+ \ell^-\) width:

\[ R_\ell = \frac{\Gamma_h}{\Gamma_\ell}\,, \]

where \(\ell\) denotes a charged lepton, and lepton-flavour universality is assumed.

See also
EW_NPZff::Rlepton() and the detailed description of EW class for the inclusion of new physics contribution
