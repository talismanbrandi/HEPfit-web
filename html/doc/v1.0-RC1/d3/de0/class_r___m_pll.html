Detailed Description

A class for the binned observable ratio \(\frac {<BR>_{M \to P l_1^+l_1^-}}{<BR>_{M \to P l_2^+l_2^-}}\) in \(M \to P l^+l^-\).

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to compute the binned observable ratio \(\frac {<BR>_{M \to P l_1^+l_1^-}}{<BR>_{M \to P l_2^+l_2^-}}\) in \(M \to P l^+l^-\) in terms of the binned observables \(<\Gamma'>_{M \to P l_i^+l_i^-}\):

\[ <R>_{M \to P l^+ l^-}= \frac {<\Gamma'>_{M \to P l_1^+l_1^-}}{<\Gamma'>_{M \to P l_2^+l_2^-}} \,. \]
