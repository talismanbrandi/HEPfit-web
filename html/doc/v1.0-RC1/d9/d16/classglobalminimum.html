Detailed Description

An observable class to prevent the occurence of a second deeper vacuum.

Author
HEPfit Collaboration
Copyright
GNU General Public License

This class is used to assure that our vacuum is the global minimum of the potential. Modified discriminant from [39]. (Scaled by a factor \(10^{-7}\) and some always positive factors removed.)
