Detailed Description

A model class for new physics in the form of oblique and \(Zb\bar{b}\)-vertex corrections.

Author
HEPfit Collaboration
Copyright
GNU General Public License

This is a Model class containing the necessary functions to compute new physics contributions to the electroweak precision observables with the Peskin-Takeuchi oblique parameters [165], [166] and with additional shifts to the left-handed and right-handed \(Zb\bar{b}\) vertices. The effective \(Zb\bar{b}\) couplings involve new physics contributions both from the oblique corrections and from the vertex corrections:

\[ \mathcal{L}_{\mathrm{eff}} = \frac{e}{2s_W c_W}\, Z_\mu\, \bar{b} \left[ g_{R,\mathrm{eff}}^b\, \gamma_\mu (1 + \gamma_5) + g_{L,\mathrm{eff}}^b\, \gamma_\mu (1 - \gamma_5) \right] b \]

where the effective couplings are given by

\[ g_{R,\mathrm{eff}}^b = g_{R,\mathrm{SM}}^b + g_{R,\mathrm{SM}}^b \frac{\alpha(M_Z^2)\, T}{2} + g_{R,\mathrm{SM}}^b \frac{\alpha(M_Z^2)\left( S - 4\,c_W^2s_W^2\, T \right)} {4s_W^2\,(c_W^2-s_W^2)} + \delta g_{R,\mathrm{vert}}^b\,, \\ g_{L,\mathrm{eff}}^b = g_{L,\mathrm{SM}}^b + g_{L,\mathrm{SM}}^b \frac{\alpha(M_Z^2)\, T}{2} + g_{R,\mathrm{SM}}^b \frac{\alpha(M_Z^2)\left( S - 4\,c_W^2s_W^2\, T \right)} {4s_W^2\,(c_W^2-s_W^2)} + \delta g_{L,,\mathrm{vert}}^b\,. \]

Initialization

After creating an instance of the current class with the constructor NPSTUZbbbarLR(), it is required to call the initialization method InitializeModel(). In the Monte Carlo run, the constructor as well as the initialization method are called in InputParser::ReadParameters().

Model parameters

The model parameters of NPSTUZbbbarLR are summarized below:

Label LaTeX symbol Description
deltaGLb \(\delta g_{L,\mathrm{vert}}^b\) New physics vertex corrections to \(g_{L}^b\).
deltaGRb \(\delta g_{R,\mathrm{vert}}^b\) New physics vertex corrections to \(g_{R}^b\).

Model flags

There is no model flag in the current class.
